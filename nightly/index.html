    Quick Install¶

    Build Configuration¶

    project_name is primarily used as a build plugin. The preferred pathway is to configure it in your pyproject.toml:

    [build-system]
requires = ["hatchling", "project_name"]
build-backend = "hatchling.build"

[tool.hatch.version]
source = "project_name"

    For advanced installation options, Setuptools alternatives, and step-by-step onboarding, see the Installation Guide.
